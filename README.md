### What is this?
+ My first big MERN-stack project
+ Online-shop of stickers
+ Portfolio project

<hr>

### How To Use
+ Clone this repo to your computer
+ "npm install" to install necessary modules
+ "npm run dev" to run front-end and back-end(on port 3000 and 5000)
